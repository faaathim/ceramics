# Handmade Ceramic E-Commerce

A Django-based e-commerce web application for selling handmade ceramic products.

## Tech Stack

Python (Django) • PostgreSQL • HTML • CSS

## Features

Product listing • Cart system • User authentication • Order management • Admin panel

## Setup

git clone <repo-url> && cd <repo>
pip install -r requirements.txt
python manage.py migrate && python manage.py runserver
